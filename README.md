Setup project for a multiplayer browser game, based in the brazilian version of the card game [Truco](https://en.wikipedia.org/wiki/Truco). It will be implemented using [Nest](https://github.com/nestjs/nest). 

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
